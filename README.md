# 🍽️ Restaurante donde Leo - Sitio Web

¡Bienvenido al repositorio del sitio web oficial del **Restaurante donde Leo**! Este proyecto busca ofrecer una experiencia digital intuitiva para que los comensales conozcan nuestro menú, instalaciones y puedan realizar reservas de forma sencilla.

## 🚀 Estado del Proyecto
Actualmente el proyecto se encuentra en **fase de desarrollo inicial**. 
- [x] Estructura HTML básica.
- [x] Archivo de estilos CSS inicial.
- [ ] Diseño responsivo (móvil/escritorio).
- [ ] Formulario de contacto/reservas funcional.

## 🛠️ Tecnologías Utilizadas
Este sitio está construido utilizando tecnologías web estándar:
* **HTML5:** Para la estructura y semántica del contenido (`Home.html`).
* **CSS3:** Para el diseño, colores y estilos visuales (`estilo_home.css`).

## 📂 Estructura del Repositorio
```text
├── Home.html          # Página de inicio del restaurante
├── estilo_home.css    # Estilos visuales del sitio
└── README.md          # Documentación del proyecto
```

## 💻 Cómo Visualizar el Proyecto Localmente

Si quieres ver el proyecto en tu navegador desde tu computadora, sigue estos pasos:

1. **Clona este repositorio:**
   ```bash
   git clone https://github.com
   ```
2. **Navega a la carpeta del proyecto:**
   ```bash
   cd "restaurante donde Leo"
   ```
3. **Abre el archivo principal:**
   Dobles clics en `Home.html` para abrirlo en tu navegador web favorito (Chrome, Edge, Firefox, etc.).
